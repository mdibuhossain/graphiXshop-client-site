# graphiXshop
## [Live Site](https://graphix-shop.web.app/)


### `Technology stack`

- HTML
- Tailwind CSS
- ReactJS
- SwiperJS
- React-vanilla-Tilt
- React Router DOM
- Firebase Auth
- NodeJS
- mongoDB


### `What it does?`
- Admin can add new product
- Admin can delete any product
- Admin can make new admin
- Admin can manage all orders
- Normal user can buy product
